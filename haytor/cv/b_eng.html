Final Year Project

‘Investigating the use of genetic algorithms to program PLD’s’

The purpose of this project was to investigate the use of genetic algorithms to design the internal connections, or fusemap, required to program a PLD1 to perform a pre-determined task. This is traditionally achieved by manually describing the circuit in a programming language, e.g. PLPL2, or deriving the logic equations and using an application to produce a fusemap.

By supplying the project software3 with the required inputs and outputs, the genetic algorithm was able to evolve a gene or solution through a process of ‘survival of the fittest’. The project was successful in producing a genetic algorithm system able to design solutions for both combinational and sequential problems.

In addition to the main project software, a PLD simulator, several programs4 were produced to test the validity of the genetic algorithm system. These consisted of:

  • a program to extract the product terms for a gene, enabling the logic equations to be derived and hence the fusemap to be manually checked.
  • a fixed gene program to test the accuracy of the PLD simulation code.
  • a random gene generator designed to replace the genetic algorithm and act as a control, illustrating the superiority of the evolutionary method.
