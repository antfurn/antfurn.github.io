Telsis Ltd, Fareham. (1995-1996)

  • Composed a program in Visual Basic 3 to control a special label printer, specifically to print labels for PROMs detailing: version, checksum, build number, variants and communication type.
  • Produced a technical brief describing the program code to enable further development by other programmers.
  • Assisted in Stores department. Functions included storing, issuing and packaging electrically sensitive components and equipment.
  • Authored a GUI in Visual Basic 4 to accelerate the programming of PROM’s via a Lloyd programmer. This included serial and parallel port communication with the Lloyd programmer. The GUI had to be capable of accepting a large number of options from the user, selecting the correct master PROM software and setting the values in the Lloyd programmer. This software also interfaced with the above mentioned label printing program to print the correct label.
  • Assisted in the writing of a Visual C++ DLL for use in cataloguing Word documents from multiple users.
